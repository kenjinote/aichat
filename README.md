# aichat

チャットのようにメッセージを入力すると、AI がメッセージに対する返答を返してくれるアプリです。

内部的には、インターネットを介して OpenAI の API を使用しています。

API の呼び出し回数には制限があるため、制限を超えると動作しなくなることがあります。
